This is a CMake project and all the files generated by the build are under the "build" subdirectory.
You **MUST** avoid accidentally globbing into the "build" directory unless strictly necessary.
Main subdirectories in this workspace:
- "src" contains the main product source code.
- "tests" contains the unit tests.
- "data" contains some files used at runtime, like system prompts.
- "src/Tools" contains the implementation of AI agent tools.
All source files must end on a newline.
